## 🛠 기술 스택 (Tech Stack)
이 프로젝트는 아래의 주요 기술 스택을 사용하여 구축되었습니다.
* **프론트엔드 (Frontend):** React
* **스타일링 (Styling):** Tailwind CSS (유틸리티 클래스 기반)
* **아이콘 라이브러리:** Lucide React
* **패키지 매니저:** pnpm

## 🚀 설치 및 실행 방법 (Getting Started)

이 프로젝트는 `pnpm`을 패키지 매니저로 사용합니다.

### 1. 환경 준비 및 저장소 복제 (Clone)

먼저 **Node.js**와 **pnpm**이 설치되어 있는지 확인합니다.

```bash
# pnpm이 없다면 먼저 설치 (있는 경우 생략)
npm install -g pnpm

# GitHub 저장소 복제
git clone [당신의 저장소 URL]
