### 환경 요구 사항
* Node.js (LTS 버전 권장)
* pnpm (패키지 관리자로 사용)
### 단일 명령어로 설치 및 실행
**저장소 복제, 의존성 설치, 그리고 개발 서버 실행까지 다음 명령어 블록 하나로 처리할 수 있습니다.**
```bash
# 1. 저장소 복제 (경로는 실제 GitHub URL로 대체하세요)
git clone [당신의 저장소 URL]
cd busan-job-matching

# 2. 의존성 설치 및 서버 실행
# pnpm install 후 곧바로 pnpm run dev 실행
pnpm install && pnpm run dev
